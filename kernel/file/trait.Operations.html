Expand description

Corresponds to the kernel’s struct file_operations.

You implement this trait whenever you would create a struct file_operations.

File descriptors may be used from multiple threads/processes concurrently, so your type must be Sync. It must also be Send because Operations::release will be called from the thread that decrements that associated file’s refcount to zero.
