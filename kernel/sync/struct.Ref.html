pub struct Ref<T: RefCounted + ?Sized> { /* fields omitted */ }

A reference-counted pointer to an instance of T.

The reference count is incremented when new instances of Ref are created, and decremented when they are dropped. When the count reaches zero, the underlying T is also dropped.

Invariants

The value stored in RefCounted::get_count corresponds to the number of instances of Ref that point to that instance of T.
